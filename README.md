# Sistema de Gestao de Estoque de Carros
# Sobre o Projeto
Este projeto foi criado para ajudar lojas de carros a organizarem melhor o que tem para vender. O objetivo e trocar as anotacoes manuais ou planilhas por um programa que guarda tudo de forma organizada no computador.
# O que o programa faz?
-Controle de carros: permite cadastrar novos veiculos, ver a lista completa, atualizar informacoes e remover itens do estoque.

-Busca de veiculos: ajuda a encontrar um carro especifico rapidamente dentro do sistema.

-Registro de vendas: guarda os dados de cada venda realizada, vinculando o cliente ao carro vendido e atualizando a quantidade disponivel no patio.

-Armazenamento seguro: todas as informacoes sao salvas em um banco de dados para que os dados nao sejam perdidos quando o programa for fechado.

# Como sera feito
O progama sera desenvolvido usando a linguagem Java e funcionando por comandos de texto no terminal. Ele foi organizado de uma forma que separa as regras da parte visual, o que deixa o sistema mais seguro e facil de mexer.
